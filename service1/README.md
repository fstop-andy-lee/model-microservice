# Test URL

## 以 Event 方式呼叫

  http://localhost:8080/api/event

## 以 Event 方式呼叫造成 saga 補償

  http://localhost:8080/api/event?testCase=2

## 以 Restful 方式呼叫

  http://localhost:8080/api/saga

## 以 Restful 方式呼叫造成 saga 補償

  http://localhost:8080/api/saga?testCase=2

## 查詢相關資料的 URL

### 查詢主檔

  http://localhost:8080/masters
  
### 查詢暫存檔

  http://localhost:8081/temps
  
### 查詢 service log

  http://localhost:8080/log0s
  http://localhost:8081/log1s
  http://localhost:8082/log2s
